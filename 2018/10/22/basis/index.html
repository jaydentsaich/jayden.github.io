事件處理

發表於 2018-10-22 | 分類於 javascript 基礎 
<!--<div>
    <button id="obt" onclick="demo()">按鈕</button>
</div>

<script>
    function demo () {
        alert('hello, html事件處理')
    }
</script>-->

<!--
    2.DOM 0級操作
        直接使用document.getElementById獲取button元素,並賦值給一個名稱為obt的變量,
        接著使用變量名.onclick的方法調用函數,優點是當函數名需要改動時不要回頭更改html裏
        的東西,缺點是一個變量只能調用一個函數,若一次調用多個函數,則後面調用的函數,會把前面
        的函數覆蓋掉,下面代碼只會跳出'這是DOM 0級操作2'
-->

    <!--<div>
        <button id="obt">按鈕</button>
    </div>

    <script>
        var obt = document.getElementById('obt');
        obt.onclick = function demo1 () {
            alert('這是DOM 0級操作1');            //這個會被覆蓋掉
        }

        obt.onclick = function demo2 () {
            alert('這是DOM 0級操作2');            //只會SHOW出這個
        }

    </script>-->

<!--
    3.DOM 2級操作
        *變量名.addEventListener('事件', 函數名)
         這個方法可直接執行事件觸發後調用的函數,並且不會有覆蓋的問題,若有多
         個函數,會依序處理掉。

        *變量名.removeEventListener('事件', 函數名)
         若要刪除addEventListener增加的函數,則使用這個語法。

-->

<!--<div>
    <button id="obt">按鈕</button>
</div>

    <script>
        var obt = document.getElementById('obt');
        obt.addEventListener('click', demo1);
        obt.addEventListener('click', demo2);
        obt.addEventListener('click', demo3);

        function demo1 () {
            alert('這是DOM 2級操作1');
        }
        function demo2 () {
            alert('這是DOM 2級操作2');
        }
        function demo3 () {
            alert('這是DOM 2級操作3');
        }

        obt.removeEventListener('click', demo2);

    </script>-->

        <!--
            事件處理必須考慮到瀏覽器版本的兼容問題,當使用者的瀏覽器是IE <= 8.0,
            並不支援addEventListener 及 removeEventListener,而必須使用attachEvent 及
            detachEvent,甚至有些更舊的瀏覽器只支援Dom 0級操作,這個時候可以使用下面的邏輯
            來寫程序
        -->

        <div>
            <button id="obt">按鈕</button>
        </div>

        <script>
            var obt = document.getElementById('obt');

            if (obt.addEventListener) {                 //如果用戶端的瀏覽器支援addEventListener,則判斷為真
                obt.addEventListener('click', demo);        
            } else if (obt.attachEvent) {                //若用戶端不支援addEventListener,則判斷是否支援attachEvent
                obt.attachEvent('onclick', demo);
            }else{                                        //如果都不支援,則使用DOM 0級操作
                obt.onclick = demo();
            }



            function demo () {
                alert('hello');
            }
        </script>

</body>
